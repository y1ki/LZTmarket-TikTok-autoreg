# 🚀 TikTok Auto Registration

Автоматизированный скрипт для регистрации аккаунтов TikTok с использованием **Playwright**, **HTTP(S)-прокси** и почтовых аккаунтов **Rambler**.

Проект автоматизирует весь процесс регистрации: от генерации данных и получения кода подтверждения по IMAP до сохранения cookies и учетных данных после успешного создания аккаунта.

---

## ✨ Возможности

- 🚀 Автоматическая регистрация аккаунтов TikTok
- 🌐 Поддержка HTTP(S)-прокси с авторизацией
- 📧 Работа с Rambler Mail через IMAP
- 🔑 Автоматическое получение кодов подтверждения из почты
- 🤖 Использование Playwright Stealth для уменьшения обнаружения автоматизации
- 🧩 Обработка reCAPTCHA и hCaptcha
- 🍪 Автоматическое сохранение cookies зарегистрированного аккаунта
- 📂 Сохранение данных аккаунтов
- ⚡ Параллельная регистрация нескольких аккаунтов
- 🔄 Автоматическая смена почтовых аккаунтов и прокси
- 📝 Подробное логирование процесса
- 🎲 Генерация случайных паролей, даты рождения и User-Agent
- 🛒 Поддержка автоматической публикации аккаунтов на LZT Market *(опционально)*


## 📦 Установка

Клонируйте репозиторий:

```bash
git clone https://github.com/y1ki/LZTmarket-TikTok-autoreg
cd LZTmarket-TikTok-autoreg
```

Установите зависимости:

```bash
pip install -r requirements.txt
```

Установите браузер Chromium:

```bash
playwright install
```

---

## 🚀 Запуск

```bash
python main.py
```

---

## ⚙️ Настройка

Перед запуском заполните необходимые файлы.

### `proxies.txt`

Формат:

```text
ip:port:login:password
```

Пример:

```text
127.0.0.1:8080:user:password
```

---

### `mails.txt`

Формат:

```text
login;password
```

или

```text
email@rambler.ru;password
```

Пример:

```text
example;password123
```

или

```text
example@rambler.ru;password123
```

---

### Основные параметры

Изменяются непосредственно в начале файла `main.py`.

```python
price = 1          # Цена лота
count = 1          # Количество регистраций за цикл
count_window = 1   # Количество одновременно открытых окон
break_time = 1     # Перерыв между циклами (минуты)
```

---

## 📂 Результат работы

После успешной регистрации:

- Cookies сохраняются в папку `accounts/`;
- данные аккаунта сохраняются в `acc.txt`;
- создаётся лог работы `registration.log`;
- при наличии токена возможна автоматическая публикация аккаунта на **LZT Market**.
